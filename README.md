# POWER-BI-Project---HR-Analytics
This project showcases an interactive HR Analytics Dashboard developed using Power BI to monitor and analyze workforce data. The dashboard provides insights into employee demographics, attrition, job roles, department-wise distribution, and performance indicators.
It enables stakeholders to quickly identify trends, patterns, and potential HR challenges through dynamic visuals and filters.

Objectives

Analyze employee attrition and retention trends

Understand workforce composition and demographics

Support data-driven HR decision-making

Provide a centralized, visual overview of key HR metrics

This project highlights practical skills in Power BI, data modeling, and HR analytics.
